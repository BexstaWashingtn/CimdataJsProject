# sDrums

Webbasierte Drummachine mit Step-Sequencer.

## Hintergrund

Dieses Projekt entstand als Abschlussprojekt im Rahmen der CimData-Weiterbildung im Modul _JavaScript Aufbaukurs_.

Der Fokus lag auf der Umsetzung zentraler JavaScript-Konzepte wie modulare Struktur, Event-Handling und der Arbeit mit Audio im Browser.

Der Funktionsumfang bildet den Kern einer Drummachine ab. Einige geplante Features wurden im Rahmen der Weiterbildung nicht mehr vollständig umgesetzt.

## Features

- 16-Step Sequencer
- BPM-Steuerung
- Mehrere Sound-Channels
- Pattern-Auswahl

## Tech Stack

- Vanilla JavaScript (ES Modules)
- HTML / CSS
- Web Audio API / Howler.js
